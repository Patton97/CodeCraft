[To change the level code]
1. Navigate to the "Program" folder
2. Double-click the "package.bluej" file
3. Double-click the class you wish to change (LevelOne, LevelTwo, etc.)
4. Enter your code
5. Click “Compile” in the top left

[To run the game]
1. Navigate to the "Program" folder
2. Double-click the "package.bluej" file
3. Right-click the "Codecraft" class
4. Choose "new Codecraft()"
5. Select the level you wish to view
6. Press the play button
